## 🧑‍💻 关于我们

中国计算机学会（CCF）四川大学学生分会是由四川大学计算机相关专业的本科生、研究生自愿组成的学术性学生组织。我们旨在搭建一个开放、共享、互助的平台，连接校内计算机爱好者与行业前沿，提升同学们的技术能力与学术素养。

## ✨ 我们的活动

- **技术分享会**：定期邀请校内老师、优秀学长学姐及行业专家进行技术讲座
- **编程竞赛**：组织参与 CCSP、CCF CAT 算法精英赛等各类编程竞赛
- **学术沙龙**：分享前沿研究方向，交流论文阅读心得
- **企业参访**：走进知名互联网企业，了解行业发展动态
- **开源项目**：维护和开发开源项目，提升工程实践能力

## 🚀 加入我们

无论你是编程新手还是技术大神，只要你对计算机科学充满热情，都欢迎加入我们！

- 关注我们的 GitHub 组织，获取最新的活动信息和技术资源
- 参与我们的开源项目，贡献代码和想法
- 扫描下方二维码加入对应交流群，和志同道合的小伙伴一起学习成长
